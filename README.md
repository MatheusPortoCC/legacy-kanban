# Legacy Kanban

Technologies:
 * Ruby on Rails 7
 * Tailwind CSS
 * Stimulus

## Installation

1. Access config/application.yml
2. Set password and username for your database
3. Run `bundle install`
5. Run `rails db:create` and `rails db:migrate`
